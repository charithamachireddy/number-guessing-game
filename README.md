# Number Guessing Game (Python)

Number Guessing Game is a simple Python console application where the computer generates a random number and the user tries to guess it.

##  Objective
The main objective of this project is to understand basic Python concepts such as loops, conditional statements, user input, and random number generation.

##  Features
- Generates a random number between 1 and 10
- Allows the user to guess multiple times
- Provides hints:
  - Too High
  - Too Low
- Displays a success message when the correct number is guessed
- Beginner-friendly and easy to understand
